# Ollama Orchestrator Simplificado

Orquestador multi-agente para Ollama optimizado para GTX 1080 Ti con offload RAM.

## 🚀 Características

- **Pipeline de Código**: Generación con programador 7B + revisión con arquitecto 14B
- **Análisis de Imágenes**: Modelo de visión 11B para análisis técnico
- **Chat Interactivo**: Interfaz conversacional con detección automática de intención
- **Gestión de Archivos**: Guardado automático de bloques de código
- **Offload RAM**: Configuración optimizada para GPUs con VRAM limitada

## 📋 Requisitos

```bash
# Python 3.8+
pip install requests colorama

# Ollama instalado
# Descarga desde: https://ollama.com
```

## ⚙️ Instalación

1. **Clonar/descargar** el script

2. **Instalar dependencias**:
```bash
pip install requests colorama
```

3. **Setup inicial** (descarga modelos):
```bash
python orchestrator.py setup
```

Esto descargará automáticamente:
- `qwen2.5-coder:14b` (Arquitecto)
- `qwen2.5-coder:7b` (Programador)
- `llama3.2-vision:11b` (Visión) - opcional

## 🔧 Configuración

### Ajustar Capas GPU

Si tienes más/menos VRAM, edita en el script:

```python
GPU_LAYERS = {
    "arquitecto": 18,   # Aumenta si tienes más VRAM
    "programador": 16,
    "vision": 20,
}
```

**Guía rápida**:
- GTX 1080 Ti (11GB): valores por defecto
- RTX 3060 (12GB): +4 capas en cada modelo
- GTX 1660 (6GB): -6 capas en cada modelo
- Solo CPU: cambia todos a 0

### Cambiar Modelos

Edita la sección `MODELS` en el script:

```python
MODELS = {
    "arquitecto": "tu-modelo:tag",
    "programador": "tu-modelo:tag",
    "vision": "tu-modelo:tag",
}
```

## 🐛 Solución de Problemas

### Ollama no inicia
```bash
# Verifica instalación
ollama --version

# Inicia manualmente
ollama serve
```

### Modelo no encontrado
```bash
# Lista modelos instalados
ollama list

# Descarga modelo manualmente
ollama pull qwen2.5-coder:7b
```

### VRAM insuficiente
- Reduce `GPU_LAYERS` en la configuración
- Usa modelos más pequeños
- Cierra otras aplicaciones que usen GPU

### Error "Connection refused"
- Verifica que Ollama esté corriendo: `ollama ps`
- Reinicia Ollama: `ollama serve`

## 📊 Uso de Recursos

**Configuración por defecto (GTX 1080 Ti)**:
- Arquitecto 14B: ~3.8 GB VRAM + ~6 GB RAM
- Programador 7B: ~2.3 GB VRAM + ~2 GB RAM  
- Visión 11B: ~4 GB VRAM + ~4 GB RAM

**Total**: ~4-5 GB VRAM máximo en simultáneo + RAM adicional

## ⚡ Tips de Rendimiento

1. **Mantén solo un modelo cargado a la vez** - Los modelos se descargan automáticamente de VRAM al terminar
2. **Usa el chat interactivo** para sesiones largas - Evita recargar modelos
3. **Ajusta temperature** en `call_ollama()` según necesites más creatividad (0.7) o precisión (0.1)
4. **Cierra navegadores/juegos** cuando uses el arquitecto 14B
